** Fibonacci in Elixir

#+BEGIN_SRC elixir

defmodule Fibonacci do
  def fibonacci(0) do 0 end
  def fibonacci(1) do 1 end
  def fibonacci(n) do fibonacci(n-1) + fibonacci(n-2) end
end

#+END_SRC

** Fibonacci in Python

#+BEGIN_SRC python

def fibonacci(n):
    if n == 0: return 0
    elif n == 1: return 1
    else: return fibonacci(n - 1) + fibonacci(n - 2)

#+END_SRC
